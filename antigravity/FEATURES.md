# CodeCon

CodeCon is a social media platform for developers.

# Features

## 1. Feed

## 2. Post

## 3. Like

## 4. Retweet

## 5. User profile

## 6. Follow

## 7. Simple algorithm for feed

# Base definitions

- Text can be up to 140 characters
- Users should be authenticated to create posts, like, retweet and follow
- Users should be able to see their own profile and the profile of other users
- Users should be able to see the feed of other users
- Users should be able to see the posts of other users
- Users should be able to see the likes of other users
- Users should be able to see the retweets of other users
- Users should be able to see the followers of other users
- Users should be able to see the following of other users
- Only darkmode is supported

# Stack
